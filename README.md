# Warga
Aplikasi yang digunakan untuk mengetahui pengeluaran, pemasukan pada setiap akun yang menempati rumah pada suatu komplek perumahan.

## Library yang digunakan
[Retrofit & OkHttp3](https://github.com/square/retrofit)

[Glide (Image Processor)](https://github.com/bumptech/glide)

[Hdodenhof CircleImageView](https://github.com/hdodenhof/CircleImageView)
